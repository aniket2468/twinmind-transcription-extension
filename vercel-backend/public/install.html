TwinMind Audio Transcription Extension

Advanced Chrome extension for real-time audio transcription

⚠️ Important: This extension requires the Vercel backend to be deployed and configured with your OpenAI API key.

Installation Steps

1 Deploy Backend First

Make sure you've deployed the Vercel backend and set your OpenAI API key in the environment variables.

2 Update Extension URL

In the extension's background.js, update the baseUrl to your Vercel backend URL:

this.baseUrl = 'https://your-app.vercel.app/api';
3 Download Extension Files

Download all the extension files from this Vercel deployment:

  • manifest.json
  • background.js
  • sidepanel.html
  • sidepanel.js
  • sidepanel.css
  • offscreen.html
  • offscreen.js
  • icon16.png
  • icon32.png
  • icon48.png
  • icon128.png
4 Load Extension in Chrome

Open Chrome and go to chrome://extensions/

Enable "Developer mode" in the top right

Click "Load unpacked" and select the folder with your extension files

✅ Success! Your extension should now be installed and working with the Vercel backend.

Testing

To test if everything is working:

  1. Open the extension sidepanel
  2. Navigate to a YouTube video or any audio source
  3. Click "Refresh Audio Detection"
  4. Try starting a recording session

Support

If you encounter issues:

  • Check the browser console for error messages
  • Verify your Vercel backend is running
  • Ensure your OpenAI API key is set in Vercel
  • Check the extension's connection status
